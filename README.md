# nantotonomemo

手持ち無沙汰になった時にできる行動を毎朝自動生成するリポジトリ。

毎朝3時(JST)に Routine が起動し、`prompt.md` の指示に従って
`today.html`(今日の提案)を生成し、`log.md`(履歴)と `profile.md`
(人物像)を更新する。

## ファイル構成

- `prompt.md` — 毎朝実行されるメインロジック
- `today_template.html` — today.html の固定デザインテンプレート
  (中身の差し替えのみ許可、レイアウトは変更しない)
- `onboarding.html` — 初回のみ表示される診断フロー(15問)
- `log.md` — 日々の実行結果を蓄積するログ(無期限)
- `profile.md` — 安定的な人物像。診断結果や明示的な要望を蒸留して保持

## 運用

1. 初回のみ、`onboarding.html` を開いて診断に回答し、結果を `profile.md` に反映
2. 以後は毎朝 `today.html` が自動更新される
3. チェックボックス・反応ボタン(😊😐😕)・任意コメントを操作するだけでよい
4. 操作しなくても、翌朝のRoutineが前日の状態を自動でログに記録する
