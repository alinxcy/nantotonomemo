# persona_log(週次・人格ため込みログ / 週報)
# 週次エージェント(prompt_weekly.md)だけが【追記】するファイル。過去の記述は上書き・削除しない。
# 役割: profile.md にはまだ昇格できない"薄いシグナル/仮説"の受け皿 +
#       profile.md 変更の監査ログ + 週次の振り返り(週報)。
#
# 【記法】
#  - 各週の見出しは週次実行日(JST)。新しい週は末尾に追記する。
#  - 3ブロック構成:
#    ① 今週のサマリ  : その週の log が示したこと(3〜5行)。
#    ② profile 変更点: この回で profile.md をどう変えたか+理由(根拠日付)。変更なしは「なし」。
#    ③ 保留・観察中  : profile.md には弱い仮説プール。各項目に 根拠(log.md 日付) を付ける。
#  - profile.md は「確定した人格」だけを綺麗に保つ。しきい値未満はここ(③)に置く。
#  - ③の仮説が再現して profile.md へ昇格したら、その項目に「→ profileへ昇格(YYYY-MM-DD)」と
#    追記して消化する(履歴として行は残す)。
#  - 根拠の日付は重複させない。捏造しない(証拠のない断定は書かない)。
#  - 変化のない週は原則スキップ(profile 変更も仮説更新もない週は追記しない)。
